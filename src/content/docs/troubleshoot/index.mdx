:::tip[Troubleshooting with SplashKit?]
If you encounter issues, ensure SplashKit is properly installed, check for syntax errors in your code, and consult the documentation for common solutions.

Updating your SplashKit installation can often resolve compatibility or bug-related issues. Follow the steps in the [Updating SplashKit Guide](/installation/#updating-splashkit) to make sure you're using the latest version.
:::
